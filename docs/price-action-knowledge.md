## 二、核心哲学：什么是价格行为

### 六大核心方面

| 方面 | 核心概念 | 交易启示 |
|------|---------|---------|
| **1. 表现形式** | 任何市场/图表/周期的价格运动 | 跨市场统一，一套方法通用 |
| **2. 图表类型** | K线/成交量/Tick/Range 图均有效 | 形式不重要，逻辑才重要 |
| **3. 每个Tick都重要** | 市场无噪音，微小事件常引发大行情 | 关注关键位置的微观结构 |
| **4. 人类行为反映** | 价格行为由理性行为+基因控制，永恒不变 | 不要依赖指标，看裸图 |
| **5. 支撑阻力+趋势+管理** | 专业交易者真正的语言 | 不靠形态记忆，靠结构分析 |
| **6. 市场惯性（80%规则）** | 80%反转失败，80%突破失败 | 站在惯性一方，逆大众而行 |

## 三、市场结构分析框架

### 3.1 市场只有两种状态

```
第一步: 判断 趋势 还是 交易区间？
第二步: 如果是趋势 → 判断是 紧凑通道、宽幅通道 还是 突破？
第三步: 根据判断构建和管理交易
```

### 3.2 80%规则

- **80%的趋势反转尝试会失败** → 逆势交易是低胜率行为
- **80%的交易区间突破尝试会失败** → 不要追突破，做失败突破（Fade）
- **市场70-80%的时间处于交易区间** → 大部分时间应该短线剥头皮

### 3.3 趋势判断

| 强度 | 类型 | 特征 |
|------|------|------|
| **最强** | 尖峰/微型通道 | 连续顺势K线，无重叠，无回调 |
| **强** | 紧凑通道 | 1-2根K线小回调，不测试20 EMA |
| **中等** | 通道 | 多根K线回调，触及20 EMA但不反转 |
| **最弱** | 宽幅通道 | 回撤40-60%，更像区间而非趋势 |

**牛市中**: 更高高点+更高低点，阳线多于阴线，价格在20 EMA之上  
**熊市中**: 更低高点+更低低点，阴线多于阳线，价格在20 EMA之下

### 3.4 趋势强度检查清单

- 连续顺势K线数量：≥3根 = 强势
- 回调持续K线数：1-3根 = 强势，≥5根 = 弱势
- 顺势K线是否收在极端位置（光头 = 强势）
- 价格是否持续在20 EMA一侧
- 新高/新低是否由强势K线推动

## 四、"始终在场"（Always In）— 最核心概念

### 定义
在任何给定时刻，如果你**必须持仓**（要么多头要么空头），你会选哪个方向？答案就是**始终在场方向**。

### 判断标准

| 始终在场做多 | 始终在场做空 | 不明确（交易区间） |
|-------------|-------------|-------------------|
| 最近的强势信号是牛市 | 最近的强势信号是熊市 | 价格围绕20 EMA振荡 |
| 价格在近期范围上部 | 价格在近期范围下部 | 阴阳K线交替 |
| 最近趋势K线是阳线 | 最近趋势K线是阴线 | 无明确高低点序列 |
| 更高高点+更高低点 | 更低高点+更低低点 | 20 EMA平坦 |
| 20 EMA向上倾斜 | 20 EMA向下倾斜 | — |

### 交易应用
- **只在始终在场方向上交易**（除非满足完整反转清单）
- **绝不对抗始终在场方向**
- 不明确时使用区间策略或空仓

## 五、入场信号系统

### 5.1 H1/H2/L1/L2 计数系统（核心入场方法）

| 牛市信号 | 含义 | 可靠性 |
|---------|------|--------|
| **High 1 (H1)** | 趋势回调后第一根突破前高的K线（单腿回调） | 中等 |
| **High 2 (H2)** | H1失败后的第二次触发（双腿回调） | **最高** ⭐ |
| High 3 (H3) | 三腿回调（楔形），通常在20EMA/前摆低点 | 较高 |
| High 4 (H4) | 四腿回调，趋势可能转弱 | 低 |

| 熊市信号 | 含义 | 可靠性 |
|---------|------|--------|
| Low 1 (L1) | 趋势反弹后第一根跌破前低的K线 | 中等 |
| **Low 2 (L2)** | L1失败后的第二次触发（双腿反弹） | **最高** ⭐ |
| Low 3 (L3) | 三腿反弹（楔形），通常在20EMA/前摆高点 | 较高 |
| Low 4 (L4) | 四腿反弹，趋势可能结束 | 低 |

### 5.2 二次入场原理（为什么H2/L2胜率最高）

1. **第一次入场**（H1/L1）：初始信号，胜率仅40-50%
2. **第一次入场失败后**：被套交易者止损 → 反向压力；新交易者看到失败 → 站在对立面
3. **第二次入场**（H2/L2）：市场再次回调触发，设置经过"测试"
4. **结果**：二次入场约**60%胜率**，一次入场仅40-50%

### 5.3 入场机制

| 方式 | 适用场景 | 方法 |
|------|---------|------|
| **止损单入场**（推荐） | 趋势交易 | 信号K线高点上方1 tick 买入；低点下方1 tick 卖出 |
| **限价单入场**（高级） | 交易区间 | 在支撑位/阻力位挂限价单 |
| **市价单入场** | 强势突破 | 确认后立即市价入场 |

**入场时效**: 信号K线后一根K线未触发 → 设置取消

## 六、10大价格行为交易模式

| # | 模式 | 概率 | 核心要点 |
|---|------|------|---------|
| 1 | **主要趋势反转** | ~40% | 低风险低概率，TBTL（10根K线+2条腿），盈亏比≥2:1 |
| 2 | **最终旗形** | ~40%摆动 | 延续模式失败后的反转 |
| 3 | **突破** ⭐ | **高概率** | Brooks最爱，任何支撑/阻力突破，K线越大越可靠 |
| 4 | **H2牛旗/L2熊旗** ⭐ | **高概率** | 面包和黄油交易，快速移动 |
| 5 | **楔形（三推）** | 形态越理想越可靠 | 至少2条腿，牛市楔形顶部做空/熊市楔形底部做多 |
| 6 | **通道** | 反转>突破 | 在反转处入场，不在突破处 |
| 7 | **测量移动** | — | 腿1≈腿2，区间突破目标=区间高度 |
| 8 | **交易区间反转** | 等第二个信号更高 | 低买高卖剥头皮 |
| 9 | **开盘反转** | — | 开盘60-90分钟内反转，随后几小时摆动 |
| 10 | **磁石（支撑阻力）** | — | 概念框架，市场总在测试关键价位 |

## 七、止损与获利规则

### 7.1 止损设置

| 规则 | 内容 |
|------|------|
| **基本规则** | 止损在信号K线对立侧1 tick之外（多头在信号K线低点下方；空头在高点上方） |
| **核心原则** | 止损应设在：如果被触及，说明交易前提是错的 |
| **信号K线过大时** | ①跳过交易 ②用入场K线代替信号K线 ③用固定点数止损 |
| **移动止损** | 获利后移至盈亏平衡点，但不要在短线目标达成前过早移动 |
| **追踪止损** | 强势趋势中每个新摆动低点下方（多头）/摆动高点上方（空头） |

### 7.2 获利目标

**Brooks默认分批策略**:
```
1. 全仓位入场
2. 短线目标（~1倍风险）平掉一半
3. 剩余仓位止损移至盈亏平衡点
4. 摆动追踪剩余半仓（用摆动点或20 EMA）
5. 测量移动目标或反转信号处全部平仓
```

| 市场状态 | 操作风格 | 目标 |
|---------|---------|------|
| 强势趋势（紧凑通道） | 波段持有 | 测量移动目标 |
| 中等趋势 | 短线+部分波段 | 1倍风险平半 + 追踪剩余 |
| 宽幅通道/区间 | 仅短线剥头皮 | 通道对面 |
| 突破进行中 | 波段 | 区间高度 |

## 八、K线分析框架

### 8.1 收盘价是最重要的价格

| 收盘位置 | 含义 |
|---------|------|
| 上方25%内 | 强势看涨 |
| 上半部分 | 温和看涨 |
| 中间附近 | 中性/犹豫 |
| 下半部分 | 温和看跌 |
| 下方25%内 | 强势看跌 |

### 8.2 K线类型速查

| 类型 | 特征 | 含义 |
|------|------|------|
| **趋势K线** | 实体≥总范围50-70%，收在极端附近 | 信念坚定 |
| **十字星** | 开收接近，影线>实体 | 均衡/犹豫/过渡 |
| **外包K线** | 高低都超过前K线 | 战斗结果由收盘位置决定 |
| **内包K线** | 高低都在前K线范围内 | 收缩，预示突破 |
| **反转K线** | 明显影线+反向实体 | 拒绝某一方向 |
| **高潮K线** | 异常大（1.5-2倍均值），收极端 | 耗尽信号 |

### 8.3 信号K线质量评估

- ✅ 实体大，收盘在趋势方向极端
- ✅ 趋势方向的影线小（被拒绝的反方向小）
- ✅ 出现在正确背景中（趋势中回调末端、支撑/阻力位）
- ❌ 实体小（十字星类）= 弱势信号
- ❌ 错误位置的完美K线 = 亏损交易

## 九、交易者方程式（数学基础）

```
(P_win × Reward) > (P_loss × Risk)

其中 P_win + P_loss = 1
```

**实际应用**:
- 60%胜率 × 2:1盈亏比 = 0.6×2 - 0.4×1 = **0.8** ✅ 正期望
- 40%胜率 × 3:1盈亏比 = 0.4×3 - 0.6×1 = **0.6** ✅ 正期望
- 50%胜率 × 1:1盈亏比 = 0.5×1 - 0.5×1 = **0** ❌ 零期望

**关键**: 高概率和好盈亏比不能兼得，但组合必须为正期望。

## 十、关键交易规则（10条铁律）

1. **每笔交易前确定"始终在场"方向** — 不知道站哪边就不交易
2. **顺势交易** — 在趋势中只做顺势入场，除非满足完整反转清单
3. **等待二次入场** — H1/L1胜率40-50%，H2/L2胜率60%
4. **信号K线收盘最重要** — 牛市应收近高点，熊市应收近低点
5. **背景比单根K线更重要** — 先评估位置，再看K线外观
6. **大多数突破失败** — 不要追突破，等跟进或交易失败
7. **先管风险，再想回报** — 入场前必须知道止损位置和风险金额
8. **交易者方程式必须为正** — 不能合理估算就不要交易
9. **远离铁丝网和不明确市场** — 最好的交易者在条件差时观望
10. **每根K线都是信息** — 阅读每根K线，在提供正方程式的K线上行动

## 十一、市场状态与策略对应表

| 市场状况 | 交易类型 | 入场方式 | 目标 | 操作风格 |
|---------|---------|---------|------|---------|
| 强势趋势（紧凑通道） | 顺势回调 | H2/L2止损单 | 波段（测量移动） | 持有 |
| 中等趋势 | 顺势回调 | 20EMA处H2/L2 | 短线+部分波段 | 分批 |
| 宽幅通道 | 双向 | 通道边缘止损/限价 | 仅短线 | 快进快出 |
| 交易区间（70-80%时间） | 极端位置Fade | 支撑/阻力限价单 | 短线至对立边缘 | 低买高卖 |
| 突破进行中 | 突破回调 | 第一次回调止损 | 波段（区间高度） | 持有 |
| 高潮/耗尽 | 反转 | 极点反转K线后止损 | 短线+部分波段 | 谨慎 |

## 十二、加密货币合约交易应用要点

### 12.1 通用性原则
- 价格行为具有分形特性，Brooks体系适用于任何市场，包括BTC/ETH永续合约
- K线分析框架对1分钟到日线图均有效

### 12.2 需要注意的差异
- 加密货币波动性远大于传统期货 → 止损点数需相应调整（ATR倍数化）
- 24/7交易无收盘 → "开盘反转"模式不适用，但类似概念可用于关键时段（美股开盘、亚盘开盘）
- 资金费率影响 → 永续合约持仓需考虑资金费率成本
- 高杠杆环境 → 仓位管理比传统市场更关键

### 12.3 适合量化集成的信号

| 信号 | 量化可行性 | 建议实现方式 |
|------|-----------|-------------|
| H2/L2 回调计数 | ⭐⭐⭐ 高 | 识别回调腿数，20EMA处触发 |
| 趋势/K线强度评分 | ⭐⭐⭐ 高 | 连续顺势K线数量、收盘位置评分 |
| "始终在场"方向 | ⭐⭐⭐ 高 | EMA斜率+高低点序列+趋势K线比例 |
| 测量移动目标 | ⭐⭐⭐ 高 | 腿1长度投影 |
| 失败突破检测 | ⭐⭐ 中 | 突破后N根K线内反转回区间 |
| 楔形/三推识别 | ⭐⭐ 中 | 三次触及通道线+动量递减 |
| 信号K线质量评分 | ⭐⭐ 中 | 实体/影线比例+收盘位置+大小对比 |
| 高潮K线检测 | ⭐⭐ 中 | K线大小超过均值1.5-2倍 |
| 铁丝网检测 | ⭐⭐⭐ 高 | K线重叠度+实体占比+阴阳交替 |

## 十三、与「三位一体」策略的集成思路

### PA（价格行为）在三位一体中的角色

```
PA（Brooks体系）→ 提供：市场结构判断 + 入场时机 + 止损位置
SMC（聪明钱概念）→ 提供：流动性区域 + 订单块 + 机构意图
威科夫（Wyckoff）→ 提供：吸筹/分布阶段 + 量价确认 + 因果关系
```

### PA模块建议输出
1. **市场状态**: 趋势(牛/熊)/区间, 通道类型, 趋势强度(1-5)
2. **始终在场方向**: LONG/SHORT/NEUTRAL
3. **最近信号**: H1/H2/H3/L1/L2/L3 + 信号K线质量评分
4. **关键价位**: 最近摆动高/低点, 20EMA位置, 测量移动目标
5. **风险参数**: 建议止损位置(tick), 建议短线目标, 建议波段目标
